# TestNG Automation Testing with Java
## Project Overview

This repository contains automation testing examples using **TestNG with Java and Selenium WebDriver**.

The project demonstrates how to write automated test cases, manage test execution using TestNG annotations, and perform browser automation.

TestNG is a testing framework for Java inspired by JUnit and designed to support different types of testing such as **unit testing, functional testing, integration testing, and end-to-end testing**. 

##  Features

- Automated test execution using TestNG
- TestNG annotations for test lifecycle management
- Selenium WebDriver browser automation
- Test execution using TestNG XML
- Multiple test case execution
- Assertions for validation

  ##  Technologies Used

- Java
- Selenium WebDriver
- TestNG
- ChromeDriver
- IntelliJ IDEA / Eclipse

## TestNG Annotations Used

@BeforeMethod – Executes before each test method

@AfterMethod – Executes after each test method

@BeforeClass – Runs before the first test method in a class

@AfterClass – Runs after all test methods in a class

@Test – Defines a test case

@Parameters – Pass parameters to test methods

@DataProvider – Used for data-driven testing
